# Comment2PushDeer

将`typecho`博客的评论通过`PushDeer`推送

下载后解压，将文件夹名称改为`Comment2PushDeer`，然后移动到`usr/plugins`目录下即可

配置`PDKEY`即可


```bash
/**
 * PushDeer推送评论通知
 *
 * @package Comment2PushDeer
 * @author 马春杰
 * @version 1.0.0
```
