# Comment2PushDeer

将`typecho`博客的评论通过`PushDeer`推送



### 安装方式1：

下载后解压，将文件夹名称改为`Comment2PushDeer`，然后移动到`usr/plugins`目录下

然后打开网站后台的`插件`--`启用插件`--配置`PDKEY`即可



### 安装方式2：

通过`SSH`进入到站点的`/usr/plugins`目录下，运行：

`git clone https://github.com/ma3252788/Comment2PushDeer.git`

然后打开网站后台的`插件`--`启用插件`--配置`PDKEY`即可

这种方式比较容易更新~

以后每次更新插件，只需要`SSH`进入到`/usr/plugins/Comment2PushDeer`下，运行 `git pull origin master` 即可~
